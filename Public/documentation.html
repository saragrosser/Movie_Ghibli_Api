Studio Ghibli Movie API Documentation

This API allows users to retrieve information about Studio Ghibli movies, including titles, descriptions, release dates, and ratings. It's designed for developers looking to integrate movie data into their applications.

Request URL Full endpoint HTTP Method Response
Get a list of movies /movies http://localhost:8080/movies GET JSON list of movies
Get data about a single movie by ID /movies/{ID} http://localhost:8080/movies/2 GET JSON data of a movie
Add a movie /movies http://localhost:8080/movies POST Confirmation of movie addition
Remove a movie by ID /movies/{ID} http://localhost:8080/movies/1 DELETE Confirmation of movie deletion
Update a movie's information by ID /movies/{ID} http://localhost:8080/movies/1 PUT Confirmation of movie update

Get a List of Movies

Returns an array of movies, each with an ID, title, and year.

Endpoint: /movies

Method: GET

Response Format: JSON array of movies

          [
              {
                  "id": 1,
                  "title": "My Neighbor Totoro",
                  "year": 1988
              },
              // More movies...
          ]

Add a Movie

Adds a new movie to the collection.

Endpoint: /movies

Method: POST

Request Body Format: JSON object containing movie details 

          {
              "title": "Castle in the Sky",
              "year": 1986
          }

Response Format: JSON object of the added movie

          {
              "id": 11,
              "title": "Castle in the Sky",
              "year": 1986
          }
